# ゴミ分別けんさく（南そらち版）

長沼町・南幌町・由仁町（南空知公衆衛生組合）のごみ分別を**品名から検索**できるWebアプリです。

スマホで「これって何ゴミ？」と思ったときに、品名を入力するだけで該当する袋（黄色／赤／青／茶／灰／白／緑／水色／粗大／直接搬入）が一発でわかります。

## 特徴

- **約1100品目を収録**（南空知公衆衛生組合「ごみの分別手帖」令和7年3月改定版より）
- **あいまい検索 + 双方向部分一致 + ひらがな⇄カタカナ自動変換**（Fuse.js）
- **インクリメンタルサジェスト**（タイプするたびに候補が出る）
- **袋ごとの詳細ページ**（出し方ガイド＋該当品目一覧）
- **完全静的サイト**（サーバー不要、Fuse.js 同梱でオフラインでも動く）
- **スマホビュー最適化**（フラットデザイン）

## 使い方

ブラウザで `index.html` を開くだけ。

ローカル環境で試すなら：

```bash
python3 -m http.server 8765
# → http://localhost:8765/ をブラウザで開く
```

## ファイル構成

```
gomishiwake/
├── index.html              # メインHTML
├── THIRD_PARTY_LICENSES.md # 同梱ライブラリのライセンス
├── css/
│   └── style.css           # スタイル（フラットデザイン）
├── js/
│   ├── classification.js   # 分別区分の定義（袋色・出し方ガイド）
│   ├── data.js             # 品目データベース（約1100件）
│   ├── app.js              # 検索ロジック・ルーティング
│   └── vendor/
│       └── fuse.min.js     # 同梱した Fuse.js
├── tests/
│   └── search-smoke.test.js # 検索順位とオフライン起動のスモークテスト
└── README.md
```

## テスト

Node.js が入っていれば、追加の依存なしでスモークテストを実行できます。

```bash
node tests/search-smoke.test.js
```

確認内容:

- `でんち` 検索で `乾電池` が先頭に来ること
- `ペットボトル` の完全一致が維持されること
- Fuse.js あり/なしの両方で検索処理が動くこと
